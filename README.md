# 📘 Manual de Usuario – Talento E-Cundi

## 🧩 Descripción General
**Talento E-Cundi** es una plataforma web desarrollada para conectar a egresados de **Ingeniería de Sistemas de la Universidad de Cundinamarca** con empresas que buscan talento calificado en la región.  
El sistema optimiza los procesos de **búsqueda de empleo y reclutamiento**, promoviendo el desarrollo profesional y fortaleciendo la empleabilidad local.

Este repositorio contiene el **Manual de Usuario oficial** de la plataforma, elaborado como guía detallada para egresados, empresarios y administradores del sistema.

---

## 📄 Contenido del Manual
El documento incluye:

- **Introducción general** al sistema y sus objetivos.  
- **Registro e inicio de sesión** para egresados y empresas.  
- **Gestión de perfiles**: creación, edición y actualización de información profesional.  
- **Búsqueda y postulación de ofertas** (para egresados).  
- **Creación y administración de ofertas** (para empresas).  
- **Evaluaciones técnicas y resultados.**  
- **Interfaz de usuario e iconografía del sistema.**

## 🧠 Objetivos del Proyecto
- Conectar **talento con oportunidades laborales**.  
- Promover el **desarrollo profesional** en Cundinamarca.  
- Brindar una **experiencia digital accesible y eficiente** para empleadores y egresados.  
- Centralizar la **gestión de ofertas, postulaciones y perfiles profesionales**.
